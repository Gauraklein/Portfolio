This project was created to help visualize whether President Trump's tweeting had an effect on a select group of stocks. We spent a fair amount of time in the beginning planning how we would build this app and we utilized project management tools to keep track of each team members progress. While working on the application we had a couple of changes in scope as well as technologies used. All of these changes helped teach me how accept that not every line of code I write will be used in production. A tough lesson but a powerful one.
